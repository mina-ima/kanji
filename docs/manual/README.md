# 書き取り練習（漢字練習パッド） 取扱説明書

> 小学生が学年別の漢字を「楽しく」覚えるための、書き取り練習アプリです。

## このアプリでできること

- 小学校の学年（1年生など）を選んで、その学年で習う漢字を練習できます。
- 「れんしゅう（練習）モード」で漢字を見ながら書き取りができます。
- 「テストモード」で覚えたかどうかを確認できます。
- 画面はお子さん向けのやさしいデザイン（やわらかい色・読みやすいフォント）です。

## 想定ユーザー・利用シーン

- 漢字を覚え始めた小学生のお子さん。
- お子さんの家庭学習をサポートする保護者の方。
- 学校の宿題の前後や、すきま時間の漢字練習に使えます。

## 動作環境・前提

- パソコン・タブレット・スマートフォンのWebブラウザ（最新版を推奨）。
- 一部の機能で Gemini（AIサービス）の APIキーが必要になる場合があります（開発者がローカルで動かす場合）。
- インターネット接続。

## セットアップ

通常の利用者は、インストール不要です。ブラウザでアプリを開くだけで使えます。

開発者がローカルで動かす場合は、下記「開発者向けメモ」を参照してください。

## 使い方（ステップbyステップ）

1. アプリを開くと、最初に学年をえらぶ「ホーム画面」が表示されます。
2. 練習したい学年（例: 小学1年生）を選びます。
3. 上部の切り替えボタンで「れんしゅう」または「テスト」を選びます。
4. 「れんしゅう」モードでは、表示された漢字を見ながら書き取りの練習をします。
5. 「テスト」モードでは、覚えた漢字を確認します。
6. 別の学年に変えたいときは、左上の家（ホーム）ボタンでホーム画面に戻ります。

※ 細かな操作は画面の表示に従ってください。本書では未確認の項目は記載していません。
